# EduCraft Frontend

React 기반 사용자 인터페이스입니다. 
사용자가 직관적으로 학습 자료를 업로드하고, AI 생성 문제를 풀고, 모의고사를 응시할 수 있도록 설계되었습니다.

---

## 🔌 API 연동 (api.js)

```javascript
// 인증 관련
POST /users/signup           // 회원가입
POST /users/login            // 로그인
GET /users/info              // 사용자 정보 조회

// 자료 관리
POST /EduMaterials/upload             // 자료 업로드
GET /EduMaterials                     // 자료 목록 조회
PATCH /EduMaterials/:id               // 자료 수정
DELETE /EduMaterials/:id              // 자료 삭제

// 문제 생성 및 조회
POST /EduMaterials/questions          // 문제 생성
POST /EduMaterials/questions/save     // 문제 저장
GET /EduMaterials/questions/user      // 생성한 문제 조회
GET /EduMaterials/questions/bookmarks // 북마크된 문제 조회
DELETE /EduMaterials/questions/:id    // 문제 삭제
PATCH /EduMaterials/questions/:id/bookmark  // 북마크 토글

// 모의고사
POST /EduMaterials/mockExams/upload         // 모의고사 생성
GET /EduMaterials/mockExams                 // 모의고사 목록 조회
PUT /EduMaterials/mockExams/:id             // 모의고사 수정
DELETE /EduMaterials/mockExams/:id          // 모의고사 삭제
GET /EduMaterials/mockExams/:id/questions   // 모의고사 문제 조회
POST /EduMaterials/mockExams/:id/submit     // 답안 제출
GET /EduMaterials/mockExams/:id/results     // 결과 조회
```

## 🎨 주요 컴포넌트 역할

| 컴포넌트 | 기능 | 상태 관리 |
|---------|------|---------|
| **App.js** | 라우팅 및 인증 상태 관리 | localStorage (JWT 토큰) |
| **Tabs.js** | 탭 네비게이션 | 선택된 탭 상태 |
| **MaterialCollection** | 자료 목록 표시 및 관리 | 자료 리스트, 로딩 상태 |
| **CreateQuestionSet** | AI 문제 생성 | 생성 파라미터, 로딩 상태 |
| **MockExamTab** | 모의고사 목록 관리 | 모의고사 리스트, 로딩 상태 |
| **MockSolveModal** | 모의고사 풀이 인터페이스 | 현재 문제, 답변, 시간 |
| **Setting** | 계정 설정 | 비밀번호 입력 상태 |

## 🛠️ 기술 스택

- **React 18+**: UI 구성
- **Axios**: HTTP 클라이언트
- **CSS3**: 스타일링 (반응형 디자인)
- **LocalStorage**: 토큰 저장
- **JavaScript ES6+**: 비동기 처리

---

## 🔐 인증 흐름

```
1. 로그인 → JWT 토큰 발급 → localStorage 저장
2. 모든 API 요청 시 Authorization 헤더에 토큰 포함
3. 토큰 만료 시 로그인 화면으로 리다이렉트
4. 로그아웃 → localStorage에서 토큰 삭제
```
